## Overview

Droid Transfer connects Android devices with desktop workflows so data can move both ways without relying on cloud services. It is designed for tasks like safeguarding important conversations, keeping media collections in sync, and aligning contact records, all from an interface that scales cleanly across screen sizes and operating systems.

Release 1.80 brings protocol-native encryption to every transfer session so personal data stays protected while moving between devices. The update also includes built-in multilingual localization and 24/7 contextual help, making the app easier to adopt for users in different regions and experience levels.

## Usage

Connect your Android device with a USB cable, or place both devices on the same Wi-Fi network. Then open Droid Transfer and choose the categories you want to synchronize:

```

1. Launch Droid Transfer
2. Select "Contacts" from the main menu
3. Choose "Export to CSV" or "Sync to Desktop"
4. Confirm destination folder
5. Begin transfer
```

When moving media, the transcoding pipeline handles unsupported formats automatically so content remains playable on the receiving device.

## FAQ

**Q: How do I update from an older version?**  
A: Download the latest release and install it on top of your current copy. Your existing configuration and data will remain intact.

**Q: Can I transfer data without USB?**  
A: Yes. Droid Transfer supports local network transfers as long as both devices are connected to the same Wi-Fi network.

**Q: What encryption standard is used?**  
A: Protocol-native encryption uses AES-256 for data in transit. You can find additional details in the documentation.

**Q: The application does not detect my device.**  
A: Check that USB debugging is turned on for your Android device and that the correct drivers are installed. For network transfers, confirm both devices are on the same subnet.
